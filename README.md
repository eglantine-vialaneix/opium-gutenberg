# Welcome to the Opium-Gutenberg Project

## In this section you will find what you need to know in order to run our computational analysis

This project studies texts from the Gutenberg Project corpus and how opium is depected by various authors depending on the time period and their literary genre. The original dataset is publically available through the Gutenberg Project API, however its metadata does not contain the published dates of the documents. Another study of the Gutenberg Project, Chronoberg, shared their manual labelling of a good portion of the books.
